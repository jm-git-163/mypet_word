# 낱말 산책

시니어를 위한 한국어 낱말 퍼즐 · 무한 레벨 · 적응형 학습 동반자 앱

`DESIGN.md`(디자인 명세)와 `TECH_SPEC.md`(기술 명세)를 그대로 구현한 **동작하는 앱**입니다.
설치 없이 브라우저에서 열리고, 휴대폰 홈 화면에 추가하면 앱처럼 쓸 수 있으며, **인터넷 없이 100% 동작**합니다.

---

## 실행 방법

### 어르신께 드릴 때 (가장 쉬운 방법)

1. 이 폴더를 웹에 올립니다 (GitHub Pages, Netlify, Vercel 등 무료 호스팅이면 충분합니다).
2. 주소를 휴대폰에서 엽니다.
3. 브라우저 메뉴에서 **「홈 화면에 추가」**를 누릅니다.
4. 이제 바탕화면 아이콘으로 앱처럼 실행되고, 비행기 모드에서도 그대로 동작합니다.

### 내 컴퓨터에서 확인할 때

```bash
cd 낱말산책
python -m http.server 5178
# 브라우저에서 http://localhost:5178 열기
```

> `index.html`을 더블클릭해도 열리지만, 그때는 오프라인 캐시(서비스 워커)가 동작하지 않습니다.

### 검사 돌리기

```bash
node test/selftest.js
```

한글 엔진 전수 검증, 낱말 규정 검사, 9,000스테이지 생성 시뮬레이션, 실력 추정 정확도까지 **55개 검사**를 수행합니다.

## 핵심 구조

### 1. 무한 레벨 — 레벨을 만들지 않고 계산합니다

```
시드 = hash(꾸러미판, 사용자트랙, 레벨번호)
```

레벨 100만 번도 즉시 생성됩니다. 저장되는 건 **정수 하나**(`level`)뿐입니다.
같은 레벨은 언제 어디서 열어도 같은 문제가 나오고, 트랙이 다르면 다른 문제가 나옵니다.

**마당(10걸음) → 동네(100걸음) → 여정(1,200걸음)** 으로 리듬을 주고,
마당 안 7번째 자리는 항상 **되새김**이 오도록 고정했습니다.

### 2. 주력 모드: 가로세로 낱말 (레벨의 60%)

낱말들이 서로 글자를 나누어 물린 판을 매 레벨 새로 짭니다.
아래 쟁반의 글자를 눌러 채우고, 지금 채우는 낱말의 **뜻이 단서로** 보입니다.

- 판 크기 최대 7×8, 낱말 4~7개. **모든 낱말은 반드시 다른 낱말과 물려 있습니다**(따로 노는 낱말 금지)
- 실력에 따라 자동 조절: 낱말 수, 미리 채워 주는 칸 수, 헷갈림용 덤 글자 수
- 「도와주세요」는 지금 낱말의 빈 칸을 하나씩 채워 줍니다
- 나머지 40%는 뜻 고르기 · 첫소리 찾기 · 사자성어 빈칸 · 되새김이 번갈아 나옵니다

### 3. 적응 난이도 — 너무 쉽지도 어렵지도 않게

- 실력 θ와 낱말 난이도 b를 같은 눈금(기준 1200)에 둡니다.
- 목표 정답률 **80%**가 되는 난이도를 골라 냅니다. `b = θ − 241`
- 슬롯 추첨: 자신감 15% / 주력 55% / 도전 25% / 탐색 5%
- **앱을 켠 뒤 첫 두 문항은 반드시 쉬운 문제**입니다. 켜자마자 틀리는 경험을 막습니다.
- **모드 난이도 보정**: 같은 낱말이라도 ‘직접 떠올리기’와 ‘넷 중 고르기’는 다릅니다.
  이걸 반영하지 않으면 사자성어·속담 문제만 유난히 어려워집니다.

측정 결과(진짜 실력을 고정하고 앱이 찾아내게 함):

| 진짜 실력 | 앱의 추정 | 정답률 |
|---|---|---|
| 1100 | 1098 | 70% |
| 1300 | 1272 | 78% |
| 1500 | 1488 | 84% |

모드별 정답률: 글자모아 73% · 뜻고르기 75% · 첫소리 76% · 되새김 73% · 속담 69% · 사자성어 68%

### 4. 기억 — 공부처럼 느껴지지 않게

FSRS(안정성 S / 난이도 D / 인출확률 R) 모델을 온디바이스로 구현했습니다.
**“거의 잊을 뻔한” 구간(R = 0.55~0.96)의 낱말만** 골라 퍼즐 문항에 자연스럽게 섞습니다.
따로 복습 메뉴도 있지만, 주된 통로는 퍼즐 안입니다.

### 5. 어르신을 위한 배려 (동작하는 것들)

- 버튼 최소 **64×64px**, 사이 간격 16px 이상
- 글씨 **5단계**(1.0~1.75배) — 1.75배에서도 겹침·잘림 **0건** (측정 완료)
- **오터치 방지**: 화면이 바뀐 직후 0.32초 손길을 받지 않고, 같은 자리 연속 터치를 무시합니다
- **끌기 대신 누르기**: 모든 조작을 톡톡 눌러서 할 수 있습니다
- **틀려도 붉은 X·경고음·진동이 없습니다.** 강아지가 고개를 갸웃할 뿐입니다
- 세 번 틀리면 **앱이 먼저** 도와드립니다
- 「오늘은 넘어가기」는 **언제나 무료**입니다. 막히는 구간이 없습니다
- 타이머 없음 · 목숨 없음 · 연속 출석 압박 없음 · 광고 없음
- 강아지 유대는 **성적이 아니라 함께한 날 수**로만 오릅니다
- 모든 낱말에 **소리로 듣기**(기본 ‘느리게’)
- 밝게 / 어둡게 / 또렷하게 보기, 움직임 줄이기

## 알려진 한계 (정직하게)

1. **속담은 레벨에 나오지 않습니다.** 길어서 가로세로 판에 안 들어갑니다.
   대신 「배움 → 속담 마당」 코너에서 따로 즐기실 수 있습니다.
2. **낱말 수 428개** — 명세의 목표는 5만 개입니다. 지금은 씨앗 꾸러미이며,
   `js/data.js`에 줄을 추가하기만 하면 그대로 늘어납니다(구조·인덱스·생성기는 규모와 무관).
3. **아주 쉬운 쪽과 아주 어려운 쪽이 얇습니다.** 실력 θ가 900 아래이거나 1700 위인 분께는
   맞춤 난이도를 낼 낱말이 부족합니다(정답률 50~62% / 90%). 초급 2음절 낱말과
   고급 어휘를 늘리면 해결됩니다.
4. **뜻풀이·예문은 전량 자체 집필**입니다(라이선스 안전). 다만 국어 전문가 검수는 아직입니다.
4. 시니어 대면 사용성 시험(명세 §15.2, 65세 이상 8명)은 **아직 수행하지 않았습니다.**
   이건 코드로 대신할 수 없는 단계입니다.

## 음성 넣는 법

음성은 **미리 만들어 둔 mp3 파일**로만 재생합니다. API 키는 앱에 넣지 않습니다.

```bash
# 윈도우 PowerShell
$env:ELEVENLABS_API_KEY = "sk-...새로 발급받은 키..."

node tools/build-voice.js --list          # 쓸 수 있는 목소리 보기
node tools/build-voice.js --limit 5       # 다섯 개만 시험 삼아
node tools/build-voice.js --all           # 전체 (audio/ 에 저장)
```

> ⚠️ **키를 코드·깃 저장소·채팅에 적지 마세요.** 앱에 넣으면 누구나 꺼내 쓸 수 있습니다.
> 무료 플랜에서는 한국어 전용 성우(라이브러리 목소리)를 API로 쓸 수 없어 억양이 어색합니다.
> 어색하면 음성을 아예 넣지 않는 편이 낫습니다 — 앱은 음성 없이도 완전히 동작합니다.

## 낱말 늘리는 법

`js/data.js`에 한 줄씩 추가하면 됩니다. 나머지(난이도 계산, 힌트 3단계, 혼동어, 색인)는 자동입니다.

```js
// 낱말: [표기, 뜻풀이, 예문, 갈래, 등급(1초급~4희귀)]
['모과','향이 진한 노란 열매입니다.','모과차를 담가 두었어요.','먹거리',3],

// 사자성어: [표기, 한자, 뜻풀이, 예문, 등급]
['온고지신','溫故知新','옛것을 익혀 새로운 것을 안다는 뜻입니다.','…',3],

// 속담: [앞구절, 뒷구절, 뜻풀이, 등급]
['가는 말이 고와야','오는 말이 곱다','내가 먼저 곱게 말해야 …',1],
```

추가한 뒤 `node test/selftest.js`를 돌리면 규정 위반(뜻풀이 길이, 정답 누설, 중복 등)을 잡아 줍니다.
